About Firebase Chats

Created by Lovely Public School

This chat application was developed as part of our technology education initiative.

How to Use:

  1. Set your username by clicking the edit button
  2. Share your Connection ID with friends
  3. Enter a friend's Connection ID to start chatting
  4. Send messages in real-time across devices

Features:

  • Real-time messaging
  • Multiple chat support
  • Cross-device compatibility
  • Persistent chat history
